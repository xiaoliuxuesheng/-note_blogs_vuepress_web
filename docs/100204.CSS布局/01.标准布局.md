## 1.1 标准流概述

**标准流**也称为普通流或文档流，就是指标签按照HTML标签默认方式进行排列；在标准流中的元素分为行内元素、块级元素、行内块元素；其在标准流中排列特点是：

- 块级元素：在标准流中独占一行，按顺序从上到下进行排列；
- 行内块元素：在标准流中具有width和height属性，按顺序从左到右进行排列，超过父盒子宽度后自动换行；
- 行内元素：在标准流中按顺序从左到右进行排列，超过父盒子宽度后会自动换行；
