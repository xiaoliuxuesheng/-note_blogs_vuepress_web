## 3.3 颜色属性

- **作用 :**  设置文字的颜色

- **颜色取值说明**

  | 属性         | 说明                                                         |
  | ------------ | ------------------------------------------------------------ |
  | color:颜色值 | - 表示颜色的单词<br />   - 三原色取值 : **rgb(0-255)**<br />   - 透明度的三原色 : **rgba(0-255,0-1)**<br />   - 十六进制颜色表示 : **#十六进制的三原色值**<br />   - 十六进制颜色简写 : **#十六进制的三原色简写** |
