## 5.1 精灵图

* **精灵图** : 利用背景图属性和背景定位属性可以完成精灵图的设置
  * 在浏览器中的的坐标系**原点** : 左上角 (0,0)
  * 在浏览器中的的坐标系**X轴** : 从原点水平向右为正方向
  * 在浏览器中的的坐标系**Y轴** : 从原点垂直向下为正方向
