## 2.3 HTML基本结构的认识

- `<!DOCTYPE>` 	定义文档类型。
- `<html>`	定义 HTML 文档。
- `<head>`	定义关于文档的信息。
- `<title>`	定义文档的标题。
- `<body>`	定义文档的主体。
- `<meta>`	定义关于 HTML 文档的元信息。
- `<link>`	定义文档与外部资源的关系。
- `<base>`	定义页面中所有链接的默认地址或默认目标。
