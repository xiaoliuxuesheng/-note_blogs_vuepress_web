## 1.3 Sass入门

### 1. 注释

- 支持标准的CSS多行注释 `/* */`以及单行注释 `//`：多行注释会完整的编译到css文件中，单行注释不会被编译

- 多行注释中的差值语句：输出变量值

  ```scss
  $version: "1.2.3";
  /* This CSS is generated by My Snazzy Framework version #{$version}. */
  ```
